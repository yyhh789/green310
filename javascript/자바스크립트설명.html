자바스크립트란?

브라우저를 조작하여 동작하게 만드는 언어
정적인 데이터를 실시간으로 변경해서 동적구현을 완성

자바 스크립트 흐름도
이벤트발생 - 입력 - 가공 - 출력

출력방식

  • alert() - 매서드 : 경고창에 해당하는 팝업을 출력하는 명령어
  • console.log() - 브라우저의 콘솔에 텍스트나 숫자를 출력
    실제 웹사이트에 공개하기전에 프로그램을 테스트 할때 자주 사용
  • document.write() - 자바스크립트로 직접 html이나,css를 조작해서 변경하거나 새로운 요소로를 추가 할수있다.
    유연한 조작이 가능하며 실제 웹사이트,웹애플리케이션에서 많이 사용된다.

주의사항

  1. 오류를 두려워하지마세요
    console창에서 해당 오류사항을 확인할수 있습니다
  2. console창을 활용하여 매번 연산을 확인하는 습관을 들인다
  3. 주석을 활용하여 문장을 만들고 동적구현을 형성한다
    무언가를 실행하고싶을떄 00을 xx해라 라는 명령어로 만들어둔다
  4. ()괄호안에 들어가는 내용을 매개변수라 하고,객채라고 한다
    문자열 은 따음표''안에 넣고 숫자는 서식형으로 그냥 작성한다
  5. 새미클론;은 자바스크립트의 마침표와 같은 역할을 한다
    줄바꿈시 자동으로 세미클론을 찍은 역할을 한다

학습내용
  • 변수 -특정코드에서 얻은 데이터를 코드에수 다시 사용하고 싶을떄 해당데이터를 저장해두는 임시공간 정의- 대입- 읽힘, 변경
  • 조건문 - 범위를 만들어 조건에 해당할떄 변경되게 하거나 조건에서 벗어날떄 변경되지 않게 하는 구조 함수 lf문 : 만약에 ~ 해당조건이 된다면 ture, 안돼면 false이다.
  • 반복문 : 동일한처리를 컴퓨터에서 반복해서 시키는것
    공식을 만들어서 해당내역을 모두 출력한다.
