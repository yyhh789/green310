flex_레이아웃속성

flex 레이아웃은 부모자에 flex를 선언하고 자식요소를 유연하게 배치하는 속성

ul> li의 경우 ul에게 flex를 선언하면 li가 가로정렬됨 이떄 li는 items의 속성을 사용한다.

flex_container 속성
속성명 속성값과의미
display flex - 자손자를 flex로 정렬하겠다 라는 선언문
justfy-content 수평정령에 관한 설정
  • flex-start - 시작점에서부터 정렬(기본값)
  • flex-end - 끝점에서 정렬
  • center - 아이템을 부모자의 가운데 정렬
  • space-between - 처음아이템은 시작점에 붙고 마지막 아이템은 끝점에서 붗인상테에서 나머지아이템을 사이에정렬
  • space-around - 모든 아이템들의 좌우여백값 통일
aline-items 아이템들의 상하여백정렬(부모자의 높이가 지정되어있고 아이템들은 부모자 높이값보다 양이 적을떄 사용함)
  • strech - 아이템들의 위아래 맞춤(기본값)
  • flex-start - 아이템들을 위로 붙임
  • flex-end - 아이템들을 아래로 붙임
  • center - 아이템을 부모기준으로 가운데 정렬
  • space-between - 처음아이템은 시작점에 붙고 마지막 아이템은 끝점에서 붗인상테에서 나머지아이템을 사이에정렬(상하기준)
  • space-around - 모든 아이템들의 상하여백값 통일

flex-wrap 기본적으로 flex는 자손자를 한줄에만 표시하려하므로 자손자의 크기는 무시하고 공간에 가둔다.
warp / nowrap(기본값) / wrap - reverse
